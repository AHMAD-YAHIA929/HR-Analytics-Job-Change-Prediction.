
# 🤖 Smart Recruitment Assistant & HR Analytics System
## 📌 About The Project
An AI-driven machine learning solution designed to predict candidate job change behavior and streamline the recruitment screening process for HR teams.

## 📁 Repository Structure
* `/data`: Dataset used (`aug_train.csv`)
* `/presentation`: Project presentation slides
* `/report`: Detailed documentation and technical report
* `HR_Analytics_Notebook.ipynb`: Complete Jupyter/Kaggle notebook code

## 🛠️ Models & Technologies
* **Python** (Pandas, NumPy, Matplotlib, Seaborn, Scikit-Learn)
* **Algorithms:** Logistic Regression, Random Forest Classifier
* **Evaluation Metrics:** Accuracy, Precision, Recall, F1-Score, Confusion Matrix

## 📊 Key Results
* **Best Model:** Random Forest Classifier
* **Top Predictive Features:** `training_hours`, `city_development_index`, `company_size`
